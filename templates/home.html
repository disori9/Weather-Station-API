Weather Data API

URL format (specific date): http://127.0.0.1:5000/api/v1/station/yyyy-mm-dd

URL format (specific year in the station): http://127.0.0.1:5000/api/v1/year_data/station/yyyy

URL format (all data in the station): http://127.0.0.1:5000/api/v1/station

URL Example (specific date): http://127.0.0.1:5000/api/v1/10/1988-10-25

URL Example (specific year in the station): http://127.0.0.1:5000/api/v1/year_data/10/1988

URL Example (all data in the station): http://127.0.0.1:5000/api/v1/10/
